# EcoMetrics Backend Spec - Architecture and API Design

## 1 Files Sturcture
### 1.1 Raw Data Bucket
- spu-raw-data-sharepoint-sync
    - CONSUMPTION/
        - PSE_Data_2022.xlsx
        - SCL_2022.xlsx
        - AC_Facilities.xlsx
        - ...
    - REFERENCE/
        - EFID.xlsx
        - GWPs.xlsx
        - LOB_LowOrgList.xlsx

### 1.2 Results Bucket
- spu-emissions-results
    - {session_id}/
        - CALCULATED/
            - {source_name1}_emissions
            - {source_name2}_emissions
            - ...
        - VISUALIZATION/

### 1.3 Config Bucket
- sessions/
    - registry.json # Master list of all sessions
    - sess_20260125100000/
        - session_meta.json
        - sources/
            - {source_name}/
                - config.json

## 2 Core Data 
### 2.1 Standard Output Schema
Process all data to this standard schema:
| Column | Type | Required | Description |
|--------|------|----------|-------------|
| **ACCT_ID** | string | Yes | Account/Asset identifier |
| **Year** | integer | Yes | Reporting year |
| **Consumption** | float | Yes | Activity data quantity |
| **Unit** | string | Yes | Unit of consumption (gal, KWH, kg, etc.) |
| **Subtype** | string | Yes | Fuel/utility type (PSE, SCL, B20, Diesel, etc.) |
| **GHG** | string | Yes | Greenhouse gas (CO2, CH4, N2O) |
| **EF_ID** | string | Yes | Emission factor identifier |
| **GHG_MTperUnit** | float | Derived | Emission factor (mt GHG per unit) |
| **GWP** | integer | Derived | Global Warming Potential |
| **mtCO2e_calc** | float | Calculated | Final emissions (Consumption × GHG_MTperUnit × GWP) |
| **LowOrg** | string | Optional | Low-level organization code |
| **LOB** | string | Optional | Line of Business |
| **Sector** | string | Optional | Sector (Fleet, Facilities, Landfill) |
| **SourceNotes** | string | Optional | Source identification |

### 4.2 Configuration and Transformation

```bash

```

``` JSON
  "name": "Q4 2022 Analysis",

}
```
**Response**
``` JSON
{
  "success": true,
  "name": "Q4 2022 Analysis",
  "sessionId": "sess_1jd8ti", // Generated by backend
  "createdAt": "2026-01-25T10:00:00Z"
}
```
4.2.2 Save complete configuration for a data source
```bash
POST /api/sessions/{sessionId}/sources/{sourceId}/config
```

**Request**
``` JSON
{
  "sourceId": "fleet_fuel_2019_2022",
  "sourceSheet": "Sheet1",
  
  "columnMappings": {
    "ACCT_ID": {"sourceColumn": "EQ_EQUIP_NO"},
    "Consumption": {"sourceColumn": "QTY_FUEL"},
    "Subtype": {"sourceColumn": "FUEL_TYPE"},
    "Year": {"derivedFrom": "FTK_DATE", "extractType": "year"},
    "Unit": {"staticValue": "gal"},
    "Sector": {"staticValue": "Fleet"},
    "LowOrg": {"sourceColumn": "DEPT_DEPT_CODE"},
    "SourceNotes": {"staticValue": "Fleet Fuel"}
  },
  
  "efidLookup": {
    "strategy": "auto",
    "pattern": "Fuel_{Subtype}_20XX_{GHG}",
    "sectorFilter": "Fleet"
  },
  
  "transformations": [
    {
      "order": 1,
      "type": "filter",
      "config": {
        "column": "FTK_DATE",
        "operator": ">=",
        "value": "2019-01-01"
      }
    },
    {
      "order": 2,
      "type": "add_column",
      "config": {
        "newColumn": "Month",
        "expression": "extract_month(FTK_DATE)"
      }
    },
    {
      "order": 3,
      "type": "join",
      "config": {
        "joinSourceId": "loworg_reference",
        "joinSheet": "LowOrg_Table",
        "leftKey": "DEPT_DEPT_CODE",
        "rightKey": "LowOrg",
        "columns": ["LOB", "Division"]
      }
    },
    {
      "order": 4,
      "type": "group_sum",
      "config": {
        "groupBy": ["ACCT_ID", "Subtype", "Year", "Month", "LowOrg"],
        "sumColumn": "Consumption"
      }
    }
  ],
  
  "optionalColumns": ["Vehicle Location", "Vehicle Class"],
  
  "ghgExpansion": {
    "enabled": true,
    "gases": ["CO2", "CH4", "N2O"]
  }
}
```
**Response**
``` JSON
{
  "success": true,
  "sessionId": "sess_1jd8ti",
  "sourceId": "fleet_fuel_2019_2022",
  "configVersion": 1,
  "validation": {
    "status": "valid",
    "warnings": [
      "Column 'Vehicle Class' has 5% null values"
    ]
  },
  "savedAt": "2026-01-25T10:15:00Z"
}
```
**Transformation Types Reference**

| Type | Description | Config Parameters |
|------|-------------|-------------------|
| **filter** | Filter rows based on condition | `column`, `operator`, `value` |
| **add_column** | Add new column with value/expression | `newColumn`, `staticValue` OR `expression` |
| **join** | Join with another table | `joinSourceId`, `leftKey`, `rightKey`, `columns` |
| **group_sum** | Group by columns and sum | `groupBy[]`, `sumColumn` |
| **rename** | Rename column | `oldName`, `newName` |
| **drop** | Remove columns | `columns[]` |
| **concat_columns** | Concatenate column values | `columns[]`, `separator`, `newColumn` |

**Expression Functions Available:**
- `extract_year(date_column)` - Extract year from date
- `extract_month(date_column)` - Extract month from date
- `concat(col1, "_", col2)` - Concatenate values
- `upper(column)` / `lower(column)` - Case conversion
- `coalesce(col1, col2)` - First non-null value
